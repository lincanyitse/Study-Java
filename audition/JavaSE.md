# JavaSE 面试题

## 1.请说说面向对象的特征

面向对象主要包括封装、继承、多态、抽象;

+ 封装
    > 给对象提供了隐藏内部特性和行为的能力，并提供一些能被其他对象访问的方法来改变它内部的数据;

+ 继承
    > 给对象提供了从基类获取字段和方法的能力，并提供了代码的重用行，也可以在不修改类的情况下给现存的类添加新特性;
    >
    > 注意：对于接口可以多继承, 但是对于类只能是单继承;

+ 多态
    > 是编程语言给不同的底层数据类型做相同的接口展示的一种能力;

+ 抽象
    > 把想法从具体的实例中分离出来的步骤;

## 2.面向对象与面向过程的区别

+ 面向过程

  + 优点
    > 性能比面向对象高，因为类调用时需要实例化，开销比较大，容易过多消耗资源;

  + 缺点
    > 没有面向对象那样易维护、易复用、易扩展;

+ 面向对象

  + 优点
    > 易维护、易复用、易扩展，由于面向对象有封装、继承、多态性的特性，可以设计出低耦合的系统，使系统更加灵活、更加易于维护;

  + 缺点
    > 性能比面向过程低;
